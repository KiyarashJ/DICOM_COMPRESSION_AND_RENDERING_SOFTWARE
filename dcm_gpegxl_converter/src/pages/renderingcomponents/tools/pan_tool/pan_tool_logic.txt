Prerequisities: 

1_Struct : Pantool
2_Enum : Activetool
3_Context: use_context()
4_Signal : use_signal(|| Pantool {translate_x: f64, translate_y : f64, is_dragging : bool , last_mouse_x : f64 , last_mouse_y : f64})
5_Mouse Event : onmousedown , onmousemove , onmouseup , onmouseleave
6_ required: button , img , div tags


How does it work : 
when the user clicks on the pan tool button , Activetool is set to Pan , afterwards when the user clicks
and keep the left-click down is_dragging immediately to true , whenever user moves the mouse (left-click
kept down) the image also moves with the exact langtitude and latitude of mouse , at the end when the user
stops keeping left-click of mouse is_dragging turns false and image stops moving !

3_pan_set_down function : 

pub fn pan_set_down(e: MouseEvent, mut pan_signal : Signal<Pantool>) {
    let coordinates = e.page_coordinates();
    pan_signal.with_mut(|c| {
        c.is_dragging = true;
        c.last_mouse_x = coordinates.x;
        c.last_mouse_y = coordinates.y;
    });
}
    👇
    👇
    here gets the coordinations of page (not element ! we move it by screen not by element)
    turns is_dragging to true , sets coordinates of x and y to the last position of mouse
    (we need the last position due to know the amount of changed langitude and latitude
    between new position of mouse and previous position of mouse !)
    👇
    👇
    ****(user moves mouse (left-click kept down))

5_pan_set_move function : 
pub fn pan_set_move(e: MouseEvent, mut pan_signal: Signal<Pantool>){
    let is_dragging = pan_signal.read().is_dragging;
    
    if is_dragging {
        let coordinates = e.page_coordinates();
        let last_x = pan_signal.read().last_mouse_x;
        let last_y = pan_signal.read().last_mouse_y;
        
        let delta_x = coordinates.x - last_x;
        let delta_y = coordinates.y - last_y;

        pan_signal.with_mut(|c| {
            c.translate_x += delta_x;
            c.translate_y += delta_y;
            c.last_mouse_x = coordinates.x; 
            c.last_mouse_y = coordinates.y;
        });
    }
}

    👇
    👇
    first checks if is_dragging is true , if true : get the coordinates(new coordinations) ,
    read the previous position of coordinates of mouse subtract each other , set the delta
    coordinations to the translates , also the new coordinates which are gotten from coor-
    dinates variable are set to last position of mouse.
    👇
    👇
