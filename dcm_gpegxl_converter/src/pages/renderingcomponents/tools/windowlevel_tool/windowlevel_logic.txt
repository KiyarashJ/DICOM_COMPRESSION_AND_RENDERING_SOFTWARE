Prerequisities: 

1_Windowleveltool
2_Enum : Activetool
3_Signal : use_signal(|| Windowleveltool {width: f64 , center: f64, last_mouse_x: f64 , last_mouse_y : f64 , is_dragging: bool})
4_Mouse Event : onmousedown , onmousemove , onmouseleave , onmouseup
5_required : button , img , div tags


How does it work : 
User clicks on the Windowleveltool, Activetool is set to Windowlevel , afterwards user whenever 
starts to keep left-click of mouse down, is_dragging immediately turns true,
eventually windowlevel tool is gonna get changed with mouse move , 
if mouse drag is on the x axis ,windowlevel is gonna change the contrast by percentage 
if mouse drag is on the y axis ,windowlevel is gonna change the brightness by percentage 
at the end , whenever user releases left-click , is_dragging turns false and tool stops working !

3_tool_set_down function : 

pub fn tool_set_down(e: MouseEvent, tool_signal : Setdown) {
    let coordinates = e.page_coordinates();
    match tool_signal {
        Setdown::None => {},
        Setdown::Pan( mut signal) => {
            signal.with_mut(|c| {
                c.is_dragging = true;
                c.last_mouse_x = coordinates.x;
                c.last_mouse_y = coordinates.y;
            });
        },
        Setdown::Windowlevel( mut signal) => {
            signal.with_mut(|c| {
                c.is_dragging = true;
                c.last_mouse_x = coordinates.x;
                c.last_mouse_y = coordinates.y;
            });
        }
    }
}
    👇
    👇
    here gets the coordinations of page (not element ! we move it by screen not by element)
    thereafter we need to specify the type of signal , which is being given to this 
    function due to variety of functions of every tool ! if None then do nothing !
    if Pan : set the coordinates to mouse previous position, set is_dragging true,     
    if windowlevel : set the coordinates to mouse previous position, set is_dragging true,

5_tool_set_move:

pub fn tool_set_move(e: MouseEvent, tool_signal: Setdown){
        let coordinates = e.page_coordinates();
        match tool_signal {
            Setdown::None => {}
            Setdown::Pan(mut signal) => {
                let is_dragging = signal.read().is_dragging;
                if is_dragging {
                let last_x = signal.read().last_mouse_x;
                let last_y = signal.read().last_mouse_y;
                let delta_y = coordinates.y - last_y;
                let delta_x = coordinates.x - last_x;
                signal.with_mut(|c| {
                    c.translate_x += delta_x;
                    c.translate_y += delta_y;
                    c.last_mouse_x = coordinates.x; 
                    c.last_mouse_y = coordinates.y;
                });
            }
                }      
            Setdown::Windowlevel(mut signal) => {
                let is_dragging = signal.read().is_dragging;
                if is_dragging {
                    let last_x = signal.read().last_mouse_x;
                let last_y = signal.read().last_mouse_y;
                let delta_y = coordinates.y - last_y;
                let delta_x = coordinates.x - last_x;
                signal.with_mut(|c| {
                    c.width += delta_x * 2.0; 
                    if c.width < 1.0 { c.width = 1.0; }
                    c.center -= delta_y * 2.0; 
                    c.last_mouse_x = coordinates.x;
                    c.last_mouse_y = coordinates.y;
                });
                }
                
            }
        }
}
    👇
    👇
    get coordinates , checks the type of signal which is being given as parameter,
    if pan : is_dragging == true ? then get new coordinates of mouse, get the last coordinates of mouse
    subtract them , give it to translate_x , translate_y and set the new page_coordinates
    also to last_position of mouse ! 

    if windowlevel : if is_dragging == true ? then get new coordinates of mouse ,
    get the last position of mouse subtract them , then ====> 
    multiply delta_x by 2 and add it to width ,because we set the contrast by this value
    if c.width < 1.0 {c.width = 1.0} =>if the width less than 1 , the number is gonna
    be so greater , that is like 400 / 0.1 * 100 => contrast: 400000%; ! that does
    not make sanse ! 
    multiply delta_y by 2 and subtract it from center ! 
    at the end set the new coordinates to the last position of mouse ! 
    👇
    👇

7_tool_set_up function : 

pub fn tool_set_up(signal: Setdown) {
    match signal {
        Setdown::None => {}
        Setdown::Pan(mut tool_signal) => {
            tool_signal.with_mut(|f| {
                f.is_dragging = false;
            });
        }
        Setdown::Windowlevel(mut tool_signal) => {
            tool_signal.with_mut(|f| {
                f.is_dragging = false;
            });
        }
    }
}
    👇
    👇
    here just turns is_dragging to false ! when the is_dragging is false onwmousemove won't work.
