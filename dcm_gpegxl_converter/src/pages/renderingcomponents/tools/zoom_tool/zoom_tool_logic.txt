Prerequisities : 

1_Struct: Zoomtool {zoom_x: 0.0, zoom_y: 0.0, scale_amount: 1.0}
2_Enum : Activetool {None, Zoom , other tools}
3_Context : use_context()
4_Signal: use_signal(|| Zoomtool {zoom_x: 0.0, zoom_y: 0.0, scale_amount: 1.0})
5_Mouse Event : onwheel 
6_ required button , div , image tags




How does it work : 
user clicks on the zoom btn ! activetool field in struct DicomRenderData is set to Zoomtool
when zoomtool is activated , onwheel checks immediately whether value is updated. Updated ? 
send the onwheel event and the struct defined into use_signal to the zoom_set function to 
set the coordinates and induces change (zoom) !

3_zoom_set component : 


let wheel = event.delta().strip_units().y;
    let coords = event.element_coordinates();
    zoom_scale_signal.with_mut(|zoom| {
        zoom.zoom_x = coords.x;
        zoom.zoom_y = coords.y;
        if wheel < 0.0 { 
            if zoom.scale_amount < 8.0 {
                zoom.scale_amount += 0.1;
            }
        } else {
            if zoom.scale_amount > 0.5 {
                zoom.scale_amount -= 0.1;
            }
        }
    });
    👇
    👇
    extracts y coordinates (cause zoom will be happened in y), also the mouse x and y coordinates
    update use_signal coordinates by live new mouse coordinates ! also checks if mouse scrollbar
    is less than 0 and also if scale amount is less than 8.0x , zoom in +0.1 , else 
    (means if wheel >= 0.0 and scale amount is greater than 0.5) zoom out -0.1
